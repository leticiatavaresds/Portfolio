In this project, I use the Python language combined with SQLite to build a Date Warehouse for ENADE Microdata from the years 2017, 2018 and 2019 and perform data analysis followed by a prediction model.

Neste projeto, utilizei a linguagem Python em conjunto com SQLite para construir um Data Warehouse para Microdados do ENADE dos anos de 2017, 2018 e 2019, e realizei análise de dados seguida por um modelo de previsão.
