In this course offered by Johns Hopkins University on the Coursera platform, I had an introduction to the main tools and ideas in the data scientist's toolbox. The course provides an overview of the data, questions, and tools that data analysts and data scientists work with.

Neste curso oferecido pela Universidade Johns Hopkins na plataforma Coursera, tive uma introdução às principais ferramentas e ideias da caixa de ferramentas do cientista de dados. O curso fornece uma visão geral dos dados, perguntas e ferramentas com as quais os analistas e cientistas de dados trabalham.
