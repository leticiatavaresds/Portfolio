This is a website dedicated to the theatrical production "Tem Querosene na Mochila", which took place on the stage of "O Tablado" in January 2024. Crafted with pure HTML5, CSS, and JavaScript, it is a responsive webpage designed to encapsulate the rebellious essence of rock and roll.
Dynamic imagery, characterized by fluid movements and vibrant warm colors, serves as a visual homage to the production's spirited narrative. To streamline the website's creation process, I utilized the Flask library alongside Python, enabling automation. Subsequently, the Flask Frozen library was employed to generate a collection of static files, ensuring seamless hosting without the need for server-side software.

Este é um site dedicado à produção teatral 'Tem Querosene na Mochila', que ocorreu no palco de 'O Tablado' em janeiro de 2024. Desenvolvido com HTML5 puro, CSS e JavaScript, é uma página responsiva projetada para encapsular a essência rebelde do rock and roll.
Imagens dinâmicas, caracterizadas por movimentos fluidos e cores vibrantes e quentes, servem como uma homenagem visual à narrativa cheia de espírito da produção. Para simplificar o processo de criação do site, utilizei a biblioteca Flask junto com Python, permitindo automação. Posteriormente, a biblioteca Flask Frozen foi empregada para gerar uma coleção de arquivos estáticos, garantindo hospedagem sem problemas, sem a necessidade de software do lado do servidor.
