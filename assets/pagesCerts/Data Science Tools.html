In this course I was able to see more about data science tools such as Jupyter Notebooks, RStudio IDE and Watson Studio. I looked at the purpose of each tool, what programming languages they can run, their capabilities and limitations, and how data scientists currently use these tools.

Neste curso, pude ver mais sobre ferramentas de ciência de dados, como Jupyter Notebooks, RStudio IDE e Watson Studio. Analisei a finalidade de cada ferramenta, quais linguagens de programação elas podem executar, suas capacidades e limitações e como os cientistas de dados usam essas ferramentas atualmente.
